# Frontend

This folder contains the React + Vite frontend for the CS498 Final Project. It is the browser-based interface used to display and interact with the project’s tweet analysis data.

## Purpose

The frontend presents analytics results in a user-friendly interface. It is built with React, TypeScript, and Vite, and is designed to consume data from the backend service.

## Requirements

- Node.js
- npm
- Vite `^8.0.10`
- React `19.2.5`
- TypeScript `~6.0.2`

## Run Locally

From the project root:

```bash
cd frontend
npm install
npm run dev
```

## Other Commands

```bash
npm run build
npm run lint
npm run preview
```

The development server URL will be shown in the terminal after `npm run dev` starts.
